📊 Netflix Content Analysis

Análise exploratória e visualização de dados do catálogo da Netflix utilizando SQL e Power BI, com foco em geração de insights estratégicos.

🎯 Objetivo

Analisar o catálogo da Netflix para identificar:

Crescimento de conteúdo ao longo do tempo
Distribuição entre filmes e séries
Principais países produtores
Gêneros mais predominantes
🧠 Principais Insights
A Netflix apresentou crescimento acelerado a partir de 2015, indicando forte expansão do catálogo
A maior parte do conteúdo é composta por filmes, com menor participação de séries
Os Estados Unidos lideram a produção, seguidos por Índia e Reino Unido
Gêneros como Drama, Comédia e Conteúdo Internacional são os mais frequentes
🛠️ Tecnologias Utilizadas
SQL (SQLite / DBeaver)
Power BI
Power Query (tratamento de dados)
DAX (criação de métricas)
🔄 Etapas do Projeto
📥 1. Coleta de Dados
Dataset público da Netflix (Kaggle)
🧹 2. Tratamento de Dados
Separação de múltiplos países em linhas
Separação de gêneros (listed_in)
Remoção de valores nulos
Padronização de textos
🧱 3. Modelagem
Criação de medidas com DAX
Uso de DISTINCTCOUNT para evitar duplicações
Estrutura preparada para análise em BI
📊 4. Visualização (Power BI)

Dashboard interativo com:

KPIs (Total de títulos, filmes e séries)
Evolução temporal do catálogo
Top países produtores
Gêneros mais populares
